This Software is created keeping in mind the difficulties of lab assistants while installing one software over multiple devices individually.
With the help of LabAssistant one software can be installed on multiple devices simultaneously and remotely.
It requires One time setup.
One Device should act as server and others as client.
Server should have this software installed.

REQUIREMENTS>>


Windows 7/8/10
Java on Server Device
Regshot (a third party software to compare Window's Registry change before and after software install.) required on Server Device
Telnet (Telnet Server on all host devices and Telnet client on Server Device)
Nmap on server
FTP Server on Server Device.
